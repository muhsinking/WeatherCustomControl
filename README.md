# WeatherCustomControl
A custom UWP weather control using APIs from [OpenWeatherMap](https://www.openweathermap.org/) to retrieve current weather data.

This sample demonstrates:
+ How to create a custom UserControl
+ How create and use custom Acrylic brushes
+ How to add the Reveal effect to a button control
+ How to retrieve weather data from the OpenWeatherMap APIs
+ How to deserialize json data using Json.NET
